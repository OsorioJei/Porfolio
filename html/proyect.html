Aplicación Informativa sobre Energía Eólica en Colombia / Proyecto Bootcamp

Desarrollo de una aplicación web informativa enfocada en la energía eólica en Colombia, con secciones como inicio, historia de la energía eólica y ejemplos visuales de datos mediante gráficas interactivas creadas con Chart.js. El proyecto fue construido con HTML, CSS, Materialize y JavaScript. Incluye un formulario interactivo donde el usuario puede calcular su consumo energético en kWh y estimar el posible ahorro si usara energías limpias y renovables, usando fórmulas y constantes integradas en JavaScript. Los resultados se muestran dinámicamente junto con gráficos que visualizan los datos ingresados y el análisis del consumo.

Sistema de Gestión para Ferretería / Proyecto Universitario

Desarrollo de una aplicación web para la gestión integral de una ferretería, con módulos de inventario, clientes, personal, administrador, login, facturación y generación de reportes en PDF. El sistema fue desarrollado en .NET utilizando HTML, CSS y Materialize para la interfaz. Se empleó Entity Framework para la conexión con una base de datos en SQL Server. Además, se creó una API para manejar los endpoints de la base de datos y se implementó la sincronización con Microsoft Azure (cuenta gratuita temporal). El proyecto incluyó el despliegue de los archivos del frontend y backend, así como la integración y pruebas de la base de datos en la nube.
